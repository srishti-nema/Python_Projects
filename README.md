# Python_Projects
A repository containing some beginner level Python projects which I created while learning Python. 

## Hangman

This project contains a typical [Hangman](https://en.wikipedia.org/wiki/Hangman_(game)) game.

## Calculator

A desktop app created by using Tkinter module which carries out basic arithmetic operations and displays the results.

## Images Viewer

A desktop app created with the help of Tkinter and Pillow. It shows some of the images stored in a directory in a user friendly 
approach by using click buttons. 
I have created a meme viewer, for a start.
